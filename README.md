# Descrição

Repositório de estudos sobre o ciclo de vida de uma Activity no desenvolvimento Android. O projeto explora, de forma prática, como cada etapa do ciclo de vida funciona. Desde a criação até a destruição da tela. Demonstrando quando e por que métodos como onCreate, onStart, onResume, onPause, onStop e onDestroy são chamados.

O objetivo é compreender o comportamento da Activity durante diferentes situações do sistema, como abertura da tela, navegação entre Activities, rotação do dispositivo e retorno ao aplicativo, aplicando boas práticas para gerenciamento de estado e recursos.
